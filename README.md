# The Movie Cave 
This is a cave of movies where you can find the top latest as well as your favorite movies. Here you will also get to see the trailer of the film.

# 🚛 Features
- 🎞 Watch Movie Trailers
- 🔍Search Your Favourite Movies
- 🖼 Responsive UI

# 🏗 Built with
- React.js
- TMDB API 

### 🍔 Things Used
- ♟ useState 
- ♟ useEffect
- ♟ movie-trailer
- ♟ react-youtube
- ♟ react-router-dom
- ♟ axios
